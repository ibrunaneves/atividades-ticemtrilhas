DESAFIO: Uma grande empresa deseja determinar qual de seus produtos tem a preferência de seus clientes. Escreva um algoritmo para ler o tipo de produto comprado Codificado da seguinte forma: 1. Produto A; 2. Produto B; 3. Produto C; 4. FIM. Caso o usuário informe um código inválido (fora da caixa de 1 a 4, deve ser solicitado um novo código até que seja válido). O programa será encerrado quando o código informado for o número 4. Deve ser escrito a mensagem “Muito obrigado!” e a quantidade de clientes que compraram cada tipo de produto. ------------------------------------------------------------------------------------------ Preferência de Produtos ------------------------------------------------------------------------------------------
